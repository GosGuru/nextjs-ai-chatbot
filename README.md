# Chatbot DMs

Asistente conversacional para gestionar, analizar y mejorar respuestas por mensajes directos.

## Desarrollo

```bash
pnpm install --frozen-lockfile
cp .env.example .env.local
pnpm dev
```

## Despliegue

El proyecto está preparado para Vercel con Next.js y pnpm. Configurá las variables de `.env.example` y verificá el despliegue en `/api/health`.
